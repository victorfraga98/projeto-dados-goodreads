Etapa 02: base de dados

A base de dados utilizada a do site Goodreads, plataforma online com mais de 125 milhes de usurios em que os leitores publicam avaliaes e recomendaes de livros.

A Goodreads disponibiliza uma API pblica para consulta dos livros em sua base de dados (https://www.goodreads.com/api), mas ela foi desativada em dezembro de 2020, persistindo o acesso apenas aos usurios anteriores a essa data.

No site Kaggle, comunidade online de cincia de dados, foi disponibilizada por um usurio uma base de dados atualizada da data de 3 de dezembro de 2020 (https://www.kaggle.com/datasets/bahramjannesarr/goodreads-book-datasets-10m). Essa base de dados foi extrada da API do site Goodreads quando ela ainda funcionava.

aval_ano <- books_selected %>%
  filter(TotalRatings > 0 & PublishYear >= 1500 & PublishYear <= 2020) %>%
  group_by(PublishYear) %>%
  summarise(aval_media = mean(Rating),
            count = n()) %>%
  filter(count >=5) %>%
  arrange(PublishYear)

Quantidade total de avaliaes por editora

aval_editora <- books_selected %>%

Avaliao mdia por autor Avaliao qtd por autor Review qtd por autor

aval_autor <- books_selected %>%
  filter(TotalRatings > 0) %>%
  group_by(Authors) %>%
  summarise(aval_media = mean(Rating),
            count = n(),
            qtd_aval = sum(TotalRatings),
            qtd_rev = sum(CountsOfReview)) %>%
  filter(count >= 5) %>%
